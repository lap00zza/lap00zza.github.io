Simple Node Server

The following snippet creates a basic http server. const http = require("http"); const PORT=8080; const ADDR="127.0.0.1"; const server = http.createServer((req, res) => { console.log(req.url); // Steps: // Write Header > Write Body > End Response res.writeHead(200, { "Content-Type": "text/html" }); res.write("<em>Hello World</em>"); res.end(); }); server.listen(PORT, ADDR, () => { console.log(`Listening on ${ADDR}:${PORT}`); });
